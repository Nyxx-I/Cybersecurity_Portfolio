## ¿Qué es Linux?

Linux es un Sistema operativo de código abierto, basado en Unix el cual es utilizado en servidores, supercomputadoras y ciberseguridad.

 ### Características:

- Multiusuario.
- Multitarea.
- Seguro.
- Personalizable.
- Código abierto.

 ## Distribuciones

Ubuntu:
- Fácil para principiantes.
- Uso diario.

Kali Linux:
- Orientada a pruebas de seguridad.
- Incluye herramientas de hacking ético.

Debian:
- Estabilidad y servidores.


# Linux vs Window

Linux es de codigo abierto con una terminal poderosa y es usado mayormente en servidores en cambio window, tiene un codigo cerrado con una interfaz grafica y es usado en escritorios.


# Código abierto y código cerrado

## Código abierto (Open Source)

El código abierto es un modelo de desarrollo donde el código fuente de un programa está disponible para que las personas puedan estudiarlo, modificarlo y distribuir mejoras.

Características:

* Permite la colaboración de una comunidad de desarrolladores.
* Favorece la transparencia y auditoría del software.
* Puede ser adaptado según las necesidades del usuario.
* Ejemplo: Linux.

---

## Código cerrado (Closed Source)

El código cerrado es un modelo de desarrollo donde el código fuente del programa es privado y solo puede ser modificado por la empresa o desarrolladores propietarios.

Características:

* El usuario no puede revisar ni modificar el código.
* Las actualizaciones y cambios dependen del propietario del software.
* Generalmente requiere una licencia de uso.
* Ejemplo: Microsoft Windows.
